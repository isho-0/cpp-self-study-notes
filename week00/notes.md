### 📍 문법 요소 2: 예) 조건문

```cpp
if (x > 0) {
    cout << "양수입니다.";
} else {
    cout << "음수 또는 0입니다.";
}
```

- `if`, `else` 기본 구조
- 중괄호는 생략 가능하지만 가독성을 위해 사용하는 것이 좋음
